## 🛠️ ¿CÓMO FUNCIONA?

El **HF Spider-Loader** utiliza el sentido arácnido para identificar qué tipo de repositorio estás intentando bajar.

1.  **Selecciona:** Elige entre <kbd>📊 Dataset</kbd> o <kbd>🧠 Modelo</kbd>.
2.  **Lanza la Red:** Pega el link de Hugging Face (ej. `https://huggingface.co/datasets/darkred06/zurich`).
3.  **Extrae:** Selecciona tu carpeta local y presiona **🕷️ LANZAR RED**.
